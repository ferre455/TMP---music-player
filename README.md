# TMP — Music Player

TMP is a minimal terminal-based music player written in Python.  
It uses `mpv` for playback and runs entirely in your terminal.

---

## How to use TMP

### Requirements
- Python 3
- `mpv`

On Arch Linux:
```sh
sudo pacman -S mpv
````

On Debian / Ubuntu:

```sh
sudo apt install mpv
```

---

### Installation

Clone the repository:

```sh
git clone https://github.com/ferre455/TMP---music-player.git
```

Enter the project directory:

```sh
cd TMP---music-player
cd tmp
```

Run TMP:

```sh
python3 tmp
```

---

## Controls

* **↑ / ↓** – Navigate songs
* **Enter** – Play selected song
* **Space** – Pause / Resume
* **← / →** – Previous / Next song
* **q** – Quit

---

## Notes

* Music files are loaded from `~/Music`
* No playlists (by design)
* No colors
* Keyboard only
* Minimal and fast
